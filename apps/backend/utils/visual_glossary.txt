- chartType (string): can take the following values: 'line', 'bar', 'area'
- title (string): the title of the chart created, should be less than 30 characters
- description (string): a small description of the chart crated, should be less than 50 characters
- x (array): a list with the x-axis values
- xLabel (string): a string with the label of x axis
- y0 (array): a list with the y0-axis values. If multiple y axis arrays are defined, denote them with y1, y2, y3 etc.
- y0Label (string): a string with the label of y0 axis. If multiple y axis arrays are defined, denote their labels with y1Label, y2Label, y3Label etc.
- y0Color (string): a string with the color of y0 axis. If multiple y axis arrays are defined, denote their labels with y1Color, y2Color, y3Color etc.
- xAxisDateFormat (string): Date format of x axis (default is YYYY-mm-dd)
- xAxisHide (boolean): Hide the x axis (default: False)
- xAxisTickLine (boolean): Flag to define whether the tick lines should be visible
- xAxisAxisLine (boolean): Flag to define whether the y axis line should be visible
- xAxisTickMargin (integer): The distance between ticks on x axis
- yAxisHide (boolean): Hide the x axis (default: False)
- yAxisTickLine (boolean): Flag to define whether the tick lines should be visible
- yAxisAxisLine (boolean): Flag to define whether the y axis line should be visible
- yAxisTickMargin (integer): The distance between ticks on y axis
- yAxisTickCount (integer): Number of ticks on the y Axis
- dateFormatTooltip (string): Date format of the tooltip (default: YYYY-mm-dd)
- lineType (string): Type of the line, can take the values 'monotone', 'step', 'bump', 'linear' 
- hideLegend (boolean): Flag specifying whether to hide the legend (default: False)
- strokeWidth (integer): Width of the line components (default: 2)
- dot (boolean): Flag specifying whether to add a marker on the data points (default: False)
